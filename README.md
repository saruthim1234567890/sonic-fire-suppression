# 🔥 Sonic Wave Fire Suppression System

## 📌 Project Overview
This project is a software simulation of a Sonic Wave Fire Suppression System developed using HTML, CSS, and JavaScript. The simulation demonstrates how sound frequencies can affect fire intensity and gradually suppress flames without using water or chemicals.

---

## 🚀 Features
- 🔊 Sound wave animation
- 🎚️ Interactive frequency slider
- 🔥 Dynamic fire intensity reduction
- 💻 Real-time simulation
- 🌐 Responsive web interface

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- VS Code

---

## ⚙️ How It Works
1. User starts the sound system
2. Sound waves are generated from the speaker
3. Frequency is adjusted using the slider
4. Fire intensity decreases gradually
5. High frequency almost extinguishes the fire

---

## 📸 Project Output
- Low Frequency → Strong Fire 🔥
- Medium Frequency → Reduced Fire 🔥⬇️
- High Frequency → Fire Suppressed ❌🔥

---

## 🎯 Purpose
The purpose of this project is to visually simulate how sound waves can influence fire behavior using interactive web technologies.
